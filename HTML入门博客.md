# 《HTML入门笔记1》

### HTML是谁发明的

* HTML是由李爵士于1990年发明的。

### HTML 起手应该做什么

* HTML 起手先进行文档申明 ```<!DOCTYPE html>```

### 常用的表章节的标签有哪些，分别是什么意思

* h1~h6 -- 标题
* p -- 段落
* section -- 章节
* article -- 文章
* header -- 头部
* footer -- 脚部
* main -- 主体内容
* aside -- 侧边栏

### 全局属性有哪些

* class
* contentedition
* hidden
* id
* style
* tabindex
* title

### 常用的内容标签有哪些，分别是什么意思

* ol+li -- 有序列表
* ul+li -- 无序列表
* dl+dt+dd -- 自定义列表（描述列表）
* pre -- 预格式化的文本
* hr -- 分割线
* br -- 换行
* a -- 超链接
* em -- 强调
* strong -- 重要 强调
* code -- 代码文本
* quote -- 行内引用
* blockquote -- 块级引用
